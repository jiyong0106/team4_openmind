## 📍 Getting Started / 어떻게 시작하나요?

1. Repository 클론
```sh
$ git clone https://github.com/whtjdrud/team4_openmind
```

2. Dependencies 설치
```sh
$ npm install
```

3. Run 실행
```sh
$ npm start
```
